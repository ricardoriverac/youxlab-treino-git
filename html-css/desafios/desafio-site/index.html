Aprenda HTML e CSS do Zero

Aprenda a criar sites do zero com HTML e CSS, as linguagens fundamentais da web. Neste curso, você vai entender como estruturar páginas, adicionar estilos e deixar seu site bonito e funcional. Tudo explicado de forma simples, com exemplos práticos, mesmo para quem nunca programou antes. Ideal para iniciantes que querem dar os primeiros passos no desenvolvimento web.

Neste curso, abordaremos os seguintes temas:

  • Introdução ao HTML
  • Mexendo com Imagens
  • Introdução ao CSS
  • Personalização de site
  • Tabelas
