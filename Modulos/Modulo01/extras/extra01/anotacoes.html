Resumo sobre <input>, <select> e <button>

1. <input>

O <input> é usado para criar campos de entrada de dados no HTML. Ele pode ter vários tipos (tipos de input) para capturar diferentes tipos de dados.

Tipos mais comuns:

  • type="text" - Entrada de texto simples.
  • type="password" - Entrada de senha (oculta).
  • type="email" - Entrada para e-mail, com validação.
  • type="date" - Entrada para selecionar uma data.
  • type="number" - Entrada para números apenas.
  • type="checkbox" - Caixa de seleção (checkbox).
  • type="radio" - Botões de opção (radio).

Exemplo:

    
      <input type="text" name="nome" placeholder="Digite seu nome">
      <input type="password" name="senha">

2. <select>

O <select> cria uma lista suspensa (dropdown) onde o usuário pode escolher uma opção. Dentro de <select>, você usa a tag <option> para definir as opções disponíveis.

Atributos importantes:

  • name - Nome da variável que será enviada no formulário.
  • id - Identificador único do elemento (para CSS/JS).
  • multiple - Permite selecionar múltiplas opções.

Exemplo:

    
      <select name="cor" id="cores">
        <option value="vermelho">Vermelho</option>
        <option value="azul">Azul</option>
        <option value="verde">Verde</option>
      </select>

3. <button>

O <button> cria um botão clicável. Pode ser usado para enviar formulários, acionar funções JavaScript ou realizar outras ações.

Atributos importantes:

  • type - Define o tipo do botão (ex: submit, reset, button).
  • onclick - Define uma função JavaScript a ser executada ao clicar no botão.

Exemplo:

    
      <button type="submit">Enviar</button>
      <button type="button" onclick="alert('Cliquei no botão!')">Clique aqui</button>
