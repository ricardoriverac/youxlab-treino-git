Anotação sobre Propriedades CSS

1. hover

A pseudo-classe :hover é usada para aplicar um estilo ao elemento quando o usuário passa o mouse sobre ele. Isso permite criar interações dinâmicas no site, como mudar a cor de um botão ao passar o mouse.

Exemplo:

      
        button:hover {
          background-color: #45a049; /* Cor de fundo do botão ao passar o mouse */
        }

2. padding

A propriedade padding define o espaço interno entre o conteúdo de um elemento e sua borda. É usada para criar um espaço entre o texto e a borda do campo ou contêiner.

Exemplo:

      
        input {
          padding: 10px; /* Espaço interno de 10px em todos os lados */
        }

3. margin

A propriedade margin define o espaço externo entre um elemento e os outros elementos ao seu redor. Ela é usada para separar os elementos na página e garantir que eles não fiquem colados uns aos outros.

Exemplo:

      
        input {
          margin-bottom: 15px; /* Espaço inferior de 15px entre campos */
        }

4. border-radius

A propriedade border-radius é usada para arredondar os cantos de um elemento. Isso é útil para dar uma aparência mais suave e moderna aos campos de input, botões, entre outros.

Exemplo:

      
        input {
          border-radius: 5px; /* Canto arredondado com raio de 5px */
        }

Resumo

  • :hover - Aplica um estilo quando o mouse passa sobre um elemento.
  • padding - Adiciona espaço interno dentro de um elemento, separando o conteúdo das bordas.
  • margin - Adiciona espaço externo entre um elemento e outros elementos na página.
  • border-radius - Arredonda os cantos de um elemento.
